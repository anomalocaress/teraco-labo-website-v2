# Teraco Voice - インストールガイド

## システム要件

### macOS
- macOS 10.15 (Catalina) 以降
- Apple Silicon (M1/M2/M3) または Intel プロセッサ
- 500MB以上の空き容量
- マイク

### Windows
- Windows 10 (64bit) 以降
- 500MB以上の空き容量
- マイク

## インストール手順

### macOS

1. **ダウンロード**
   - お使いのMacに合わせて以下をダウンロード:
     - Apple Silicon (M1/M2/M3): `TeracoVoice-macOS-AppleSilicon.dmg`
     - Intel Mac: `TeracoVoice-macOS-Intel.dmg`

2. **インストール**
   - ダウンロードしたDMGファイルをダブルクリック
   - 表示されたウィンドウで、Teraco Voiceアイコンを「Applications」フォルダにドラッグ&ドロップ

3. **初回起動（重要）**
   
   **「開発元を確認できません」と表示される場合:**
   
   これは正常な動作です。以下の手順で起動してください：
   
   a. Finderでアプリケーションフォルダを開く
   
   b. 「Teraco Voice」アプリを **右クリック**（または Control + クリック）
   
   c. メニューから「**開く**」を選択
   
   d. 表示されたダイアログで「**開く**」ボタンをクリック
   
   ⚠️ **注意**: ダブルクリックではなく、必ず「右クリック→開く」の手順で起動してください。
   
   ✅ **2回目以降**: 通常通りダブルクリックで起動できます。
   
   💡 **なぜこの手順が必要？**
   Teraco Voiceは個人開発者によって作成されており、Appleのコード署名を取得していません。
   そのため、初回起動時のみこの特別な手順が必要となります。
   アプリ自体は安全ですので、ご安心ください。

4. **権限の許可**
   - マイクへのアクセス許可を求められたら「OK」をクリック
   - システム環境設定で権限を確認・変更できます

### Windows

1. **ダウンロード**
   - インストーラー版: `TeracoVoice-Windows-Setup.exe`
   - ポータブル版: `TeracoVoice-Windows-Portable.exe`

2. **インストール（インストーラー版）**
   - ダウンロードしたEXEファイルをダブルクリック
   - 「WindowsによってPCが保護されました」と表示された場合:
     - 「詳細情報」をクリック
     - 「実行」ボタンをクリック
   - インストール先を選択（デフォルトのままでOK）
   - 「インストール」をクリック

3. **ポータブル版の使用**
   - ダウンロードしたEXEファイルを任意のフォルダに配置
   - ダブルクリックで起動（インストール不要）

## 使い方

### 基本操作

1. **起動**
   - アプリを起動すると、画面上に小さなインジケーターが表示されます
   - 緑色のランプが点灯していれば準備完了です

2. **音声入力**
   - **Shiftキー**を長押ししながら話す
   - 話し終わったらShiftキーを離す
   - 認識された文字が自動的に入力されます

3. **インジケーターの状態**
   - **緑色のランプ**: 待機中
   - **緑色のランプ（点滅）+ 波形**: 録音中
   - **青色の回転**: 処理中

4. **インジケーターの移動**
   - インジケーターをドラッグして好きな位置に移動できます
   - 常に最前面に表示されます

### ヒント

- **短い文章**: 1〜2文ずつ入力するのがおすすめ
- **はっきりと**: できるだけはっきりと話すと認識精度が上がります
- **言い直し**: 言い間違えても、AIが文脈から推測して修正します
- **静かな環境**: 周囲の雑音が少ない環境で使用すると精度が向上します

## トラブルシューティング

### 音声が認識されない
- マイクの権限が許可されているか確認
- マイクが正しく接続されているか確認
- 他のアプリがマイクを使用していないか確認

### アプリが起動しない
- システム要件を満たしているか確認
- アプリを再インストール

### 認識精度が低い
- マイクに近づいて話す
- 周囲の雑音を減らす
- はっきりと話す

## アンインストール

### macOS
1. アプリケーションフォルダから「Teraco Voice」をゴミ箱に移動
2. ゴミ箱を空にする

### Windows
1. 「設定」→「アプリ」→「インストールされているアプリ」
2. 「Teraco Voice」を選択
3. 「アンインストール」をクリック
